<h1 align="center">  👾 Programação Competitiva </h1>


Repositório dedicado ao armazenamento de soluções de problemas de **programação competitiva**, com foco em desenvolvimento de lógica, algoritmos e estruturas de dados. Todos as implementações serão realizadas com a linguagem C++.

## Plataformas utilizadas:

- [Codeforces](https://codeforces.com/)
- [Neps Academy](https://neps.academy/)

## 👾 Tópicos:

-  Strings  
-  Busca e Ordenação  
-  Crivo de Eratóstenes  
-  Soma de Prefixos  
-  Vetor de Diferenças  
-  Vetor de Frequência  
-  Recursão
